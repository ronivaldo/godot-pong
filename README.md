# Pong (Godot)

Minimal 2D Pong clone built with Godot 4.x.

Quick start
- Open this folder in Godot 4.x and run the main scene (configured in project.godot).

Controls
- Left player: W / S
- Right player: Up / Down (arrow keys)
- `R` = restart scene, `Q` = quit, `E` = toggle trajectory lines, `B` = toggle ball debug control

Project layout
- `scenes/` — Godot scenes
- `scripts/` — GDScript game logic
- `assets/` — audio and font resources
- `specs/` — generated documentation
